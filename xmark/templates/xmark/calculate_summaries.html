    Recalculate Grade Summaries
    This utility will recalculate all grade summaries for students based on their published marks. This process will update or create grade summaries for all student-subject combinations.
    Note: This will only process marks with status "Published". Draft marks will be ignored in the calculation.
    What this process does:
    • Calculates weighted averages based on assessment type weights
    • Assigns final grades based on grade scale configuration
    • Updates grade points for each student-subject combination
    • Creates or updates grade summary records
